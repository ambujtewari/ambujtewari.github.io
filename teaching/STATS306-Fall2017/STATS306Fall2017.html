STATS 306: Introduction to Statistical Computing

Fall 2017

Class Information

  • Days & Time: Mondays & Wednesdays, 11:30 am -- 1:00 pm
  • Location: 1200 CHEM
  • Description: This is an introductory statistical computing course based on the R programming language. Topics covered include data wrangling, data visualization, basics of programming in R, and basics of statistical modeling.
  • Textbook: We will use R for Data Science by Grolemund and Wickham. It is available both as a printed book and as an online resource.
  • Canvas: You should access the Canvas class page for this course frequently. It will contain important announcements and posted homework assignments.
